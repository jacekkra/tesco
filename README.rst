Prerequisites
====

This script relies on your browser cookies to be able to make authorized requests to the server.

Firefox
----
Set ``FIREFOX_PROFILE_DIR`` in the ``.env`` file to your active user profile directory - it is the folder containing ``cookies.sqlite``.

On macOS, the path is ``~/Library/Application Support/Firefox/Profiles/\<some ID>.default``.


Other browsers
----
n/a

Running
====
Execute ``tesco.py``.
